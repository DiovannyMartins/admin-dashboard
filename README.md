# Admin Dashboard

Dashboard administrativo desenvolvido com HTML5, CSS3 e JavaScript, com painel de estatísticas, gráfico de desempenho e gerenciamento completo de usuários.

🔗 **Acesse:** [diovannymartins.github.io/admin-dashboard](https://diovannymartins.github.io/admin-dashboard/)

## 🚀 Tecnologias

- HTML5
- CSS3
- JavaScript

## ⚙️ Funcionalidades

- **Menu mobile responsivo**
- **Cards de estatísticas** (usuários, projetos, vendas, receita)
- **Gráfico de barras dinâmico**, gerado a partir de dados em JavaScript
- **Tabela de usuários com CRUD completo** — adicionar, visualizar, editar e excluir
- **Busca em tempo real** na tabela de usuários
- **Ordenação da tabela** por nome, status ou plano (clicando nos cabeçalhos)
- **Persistência de dados** via `localStorage`
- **Sistema de notificações** com dropdown e contador dinâmico
- **Totalmente responsivo** (desktop, tablet e mobile)

## ♿ Acessibilidade

- Hierarquia de títulos correta (h1 → h2 → h3)
- Contraste de cores adequado em todos os textos e badges
- Labels associados aos campos do formulário
- Navegação por teclado nos elementos interativos
